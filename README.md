# Hybrid Solar Battery Charging and Load Control System

An IoT-based system designed to monitor solar power and control battery charging and load automatically using ESP32.

## Features
- Solar voltage monitoring
- Battery voltage sensing
- Automatic charging control
- Load control using relay modules
- Real-time monitoring

## Technologies Used
- ESP32
- Embedded C
- Sensors
- Relay Modules
- IoT Concepts

## Working
The system monitors solar and battery voltage using sensors and automatically controls charging and load based on predefined conditions.
